How to Use

API Documentation

Send a request, pay a Lightning invoice, and get your result. No accounts needed.

API Endpoints

Endpoints

  • POST /v1/{api_name}/{endpoint} — submit your request and receive a Lightning invoice.
  • GET /redeem?preimage=... — submit your payment proof and receive the API response.
  • GET /api/catalog — view available APIs and pricing.

What to Expect

  • Each invoice can only be used once.
  • Your request is locked in when the invoice is created.
  • Expired invoices cannot be redeemed.
  • No signup or API key required.

How It Works

Three steps. No account needed.

1
Send your request
POST to any endpoint. You'll receive a Lightning invoice to pay.
→
2
Pay the invoice
Pay with any Lightning wallet. Your wallet will give you a preimage as proof of payment.
→
3
Get your response
Submit the preimage to the redeem endpoint. We forward your request upstream and return the result.

What is a preimage? When you pay a Lightning invoice, your wallet receives a secret called the preimage. It serves as your cryptographic proof of payment — we verify it instantly without needing any account or database lookup.

Code Examples

Ready-to-use examples for calling the API. Each one sends a request, pays the invoice, and retrieves the response.

bash — interactive (you pay manually in your wallet) 
#!/usr/bin/env bash
set -euo pipefail

API="https://alittlebitofmoney.com"

# Step 1: Send request → get 402 + Lightning invoice
echo "→ Requesting chat completion..."
RESPONSE=$(curl -s -X POST "$API/v1/openai/chat/completions" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o-mini",
    "messages": [{"role": "user", "content": "Explain Bitcoin in one sentence."}]
  }')

INVOICE=$(echo "$RESPONSE" | jq -r '.invoice')
SATS=$(echo "$RESPONSE" | jq -r '.amount_sats')
echo "→ Invoice: $SATS sats"
echo "$INVOICE"

# Step 2: Pay in your wallet. It will show you a preimage (hex string).
echo ""
echo "→ Pay this invoice in any Lightning wallet."
echo "  After paying, your wallet reveals the preimage."
read -rp "→ Paste preimage here: " PREIMAGE

# Step 3: Redeem — preimage proves you paid
echo ""
echo "→ Redeeming..."
curl -s "$API/redeem?preimage=$PREIMAGE" | jq .
python — fully automated (pays via your Lightning node) 
"""
pip install requests
Requires a Lightning node/wallet with an API that can pay invoices
and return the preimage. This example uses phoenixd — adapt pay_invoice()
for LND, CLN, Alby, or any wallet that exposes a payment API.
"""
import requests

API = "https://alittlebitofmoney.com"

# Your Lightning node (for paying invoices)
PHOENIX_URL = "http://localhost:9741"
PHOENIX_PASSWORD = "your-phoenix-password"

def pay_invoice(bolt11: str) -> str:
    """Pay a BOLT11 invoice via phoenixd. Returns the preimage."""
    r = requests.post(
        f"{PHOENIX_URL}/payinvoice",
        auth=("", PHOENIX_PASSWORD),
        data={"invoice": bolt11},
    )
    r.raise_for_status()
    return r.json()["paymentPreimage"]

# ── Step 1: Send request ────────────────────────────────
response = requests.post(
    f"{API}/v1/openai/chat/completions",
    json={
        "model": "gpt-4o-mini",
        "messages": [{"role": "user", "content": "Explain Bitcoin in one sentence."}],
    },
)

assert response.status_code == 402, f"Expected 402, got {response.status_code}"
invoice_data = response.json()
print(f"Invoice: {invoice_data['amount_sats']} sats")

# ── Step 2: Pay the invoice ─────────────────────────────
preimage = pay_invoice(invoice_data["invoice"])
print(f"Paid! Preimage: {preimage[:16]}...")

# ── Step 3: Redeem with preimage ────────────────────────
result = requests.get(f"{API}/redeem", params={"preimage": preimage})
result.raise_for_status()

data = result.json()
print(data["choices"][0]["message"]["content"])
javascript — fully automated (Node.js, pays via your Lightning node) 
/**
 * Node.js — no dependencies beyond built-in fetch (Node 18+).
 * Adapt payInvoice() for your Lightning wallet/node.
 * This example uses phoenixd.
 */

const API = "https://alittlebitofmoney.com";
const PHOENIX_URL = "http://localhost:9741";
const PHOENIX_PASSWORD = "your-phoenix-password";

async function payInvoice(bolt11) {
  const res = await fetch(`${PHOENIX_URL}/payinvoice`, {
    method: "POST",
    headers: {
      "Authorization": "Basic " + btoa(":" + PHOENIX_PASSWORD),
      "Content-Type": "application/x-www-form-urlencoded",
    },
    body: `invoice=${encodeURIComponent(bolt11)}`,
  });
  if (!res.ok) throw new Error(`Payment failed: ${res.status}`);
  const data = await res.json();
  return data.paymentPreimage;
}

async function main() {
  // Step 1: Send request → 402 + invoice
  const step1 = await fetch(`${API}/v1/openai/chat/completions`, {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
      model: "gpt-4o-mini",
      messages: [{ role: "user", content: "Explain Bitcoin in one sentence." }],
    }),
  });

  if (step1.status !== 402) throw new Error(`Expected 402, got ${step1.status}`);
  const invoice = await step1.json();
  console.log(`Invoice: ${invoice.amount_sats} sats`);

  // Step 2: Pay the invoice
  const preimage = await payInvoice(invoice.invoice);
  console.log(`Paid! Preimage: ${preimage.slice(0, 16)}...`);

  // Step 3: Redeem with preimage
  const step3 = await fetch(`${API}/redeem?preimage=${preimage}`);
  if (!step3.ok) throw new Error(`Redeem failed: ${step3.status}`);
  const result = await step3.json();

  console.log(result.choices[0].message.content);
}

main().catch(console.error);
